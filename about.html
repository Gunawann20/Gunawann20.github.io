About

Saya merupakan seorang lulusan teknik informatika yang memiliki ketertarikan dalam bidang aplikasi berbasis web dan android. Saya memiliki kemampuan dalam bahasa pemrograman seperti PHP, Java, dan Kotlin. Saya juga menguasai beberapa framework seperti Laravel, Java spring, dan Android SDK. Selama studi saya, saya telah mengikuti berbagai kegiatan untuk memperluas pengetahuan saya dalam pemrograman web maupun android. Saya memiliki pengalam dalam pengembangan web sistem informasi manufaktur PT Allure Allumunio pada kegiatan Kedaireka. Saya juga memiliki pengalaman mengikuti program Bangkit Academy 2023 dengan mengambil learning path Mobile Development, dan mengikukti program Studi Independen di Binar Academy dengan mengambil learning path Backend Java Course. Saya memiliki minat yang kuat dalam teknologi dan selalu mencari cara untuk meningkatkan keterampilan saya. Saya senang belajar hal-hal baru dan terus berinovasi untuk membuat solusi teknologi yang lebih baik dan lebih efektif. Saya percaya bahwa dengan kemampuan saya dalam web dan android programing, saya dapat memberikan kontribusi yang signiykan dalam bidang teknologi informasi dan membawa manfaat positif bagi masyarakat.
